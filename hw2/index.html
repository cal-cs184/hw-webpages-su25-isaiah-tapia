Overview

In this project we combined curve evaluation and surface modeling to explore fundamental techniques in geometry processing. Section 1 began with Bézier curves and surfaces using de Casteljau’s algorithm, which gave us an intuitive understanding of recursive interpolation in 1D and 2D. In section 2, we built a mesh editing system using the half-edge data structure, building operations such as edge flip, split, and loop subdivision for smooth upsampling. Loop sampling is used because unlike 2D images that lie on regular grids, where techniques like bilinear filtering easily perform upsampling, 3D meshes have vertices at irregular positions. This project ultimately allowed us to generate smooth, higher-resolution meshes while managing the complexity of mesh connectivity.

Part 2: Bezier surfaces with separable 1D de Casteljau

Just like how de Casteljau’s algorithm recursively interpolates between control points to evaluate a point on a Bézier curve, we can extend this idea to evaluate a point on a Bézier surface.
A Bézier surface is defined over a grid of control points (like a 2D array), with two parameters: u and v. We evaluate it in two stages:
  • First, for a fixed u, we apply the 1D de Casteljau algorithm along each row of control points. This gives us a new set of points, one per row.
  • Then, we apply the 1D de Casteljau algorithm again on this new set, but this time using the parameter v.
This gives us the final evaluated point on the surface.
